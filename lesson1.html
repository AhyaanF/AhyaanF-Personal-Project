HTML Structure

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using various elements and tags.

Basic HTML Structure


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My First Web Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is my first web page.</p>
    </body>
</html>

HTML Elements

HTML elements are the building blocks of HTML pages. An HTML element is defined by a start tag, some content, and an end tag.

Common HTML Elements:

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <div> - Container
  • <span> - Inline container

Practice Exercise

Create a simple HTML page that includes:

  • A main heading
  • Two paragraphs
  • A link to another website

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>My Practice Page</title>
    </head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is my first paragraph.</p>
        <p>This is my second paragraph.</p>
        <a href="https://www.example.com">Visit Example.com</a>
    </body>
</html>
