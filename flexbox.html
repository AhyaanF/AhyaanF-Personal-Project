CSS Flexbox

Flex Container Properties

The flex container is the parent element that holds flex items. Its properties control the layout direction and alignment of items.

Container Basics


/* Basic flex container */
.container {
    display: flex;
    /* or */
    display: inline-flex;
}

/* Direction and wrap */
.container {
    /* Main axis direction */
    flex-direction: row; /* default */
    flex-direction: row-reverse;
    flex-direction: column;
    flex-direction: column-reverse;

    /* Wrapping behavior */
    flex-wrap: nowrap; /* default */
    flex-wrap: wrap;
    flex-wrap: wrap-reverse;

    /* Shorthand for direction and wrap */
    flex-flow: row wrap;
}

/* Alignment properties */
.container {
    /* Main axis alignment */
    justify-content: flex-start; /* default */
    justify-content: flex-end;
    justify-content: center;
    justify-content: space-between;
    justify-content: space-around;
    justify-content: space-evenly;

    /* Cross axis alignment */
    align-items: stretch; /* default */
    align-items: flex-start;
    align-items: flex-end;
    align-items: center;
    align-items: baseline;

    /* Multi-line alignment */
    align-content: flex-start;
    align-content: flex-end;
    align-content: center;
    align-content: space-between;
    align-content: space-around;
    align-content: stretch; /* default */
}

Flex Item Properties

Flex items are the children of a flex container. Their properties control how they grow, shrink, and align individually.

Item Properties


/* Flex item sizing */
.item {
    /* Growth factor */
    flex-grow: 0; /* default */
    flex-grow: 1;
    flex-grow: 2;

    /* Shrink factor */
    flex-shrink: 1; /* default */
    flex-shrink: 0;
    flex-shrink: 2;

    /* Base size */
    flex-basis: auto; /* default */
    flex-basis: 0;
    flex-basis: 200px;
    flex-basis: 50%;

    /* Shorthand for grow, shrink, and basis */
    flex: 0 1 auto; /* default */
    flex: 1; /* 1 1 0% */
    flex: auto; /* 1 1 auto */
    flex: none; /* 0 0 auto */
}

/* Individual alignment */
.item {
    align-self: auto; /* default */
    align-self: flex-start;
    align-self: flex-end;
    align-self: center;
    align-self: baseline;
    align-self: stretch;

    /* Order modification */
    order: 0; /* default */
    order: 1;
    order: -1;
}

Common Flex Patterns

Learn common layout patterns that can be achieved using Flexbox.

Layout Examples


/* Navigation bar */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
}

.nav-links {
    display: flex;
    gap: 1rem;
}

/* Card layout */
.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

.card {
    flex: 1 1 300px;
    max-width: calc(33.333% - 1rem);
}

/* Holy grail layout */
.holy-grail {
    display: flex;
    flex-wrap: wrap;
    min-height: 100vh;
}

.header, .footer {
    flex: 1 1 100%;
}

.main-content {
    flex: 1 1 60%;
}

.sidebar {
    flex: 1 1 20%;
}

/* Centering content */
.center-content {
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
}

/* Equal height columns */
.columns {
    display: flex;
    gap: 1rem;
}

.column {
    flex: 1;
    padding: 1rem;
}

Responsive Flexbox

Learn how to create responsive layouts using Flexbox and media queries.

Responsive Patterns


/* Mobile-first approach */
.container {
    display: flex;
    flex-direction: column;
    gap: 1rem;
}

/* Tablet and up */
@media (min-width: 768px) {
    .container {
        flex-direction: row;
        flex-wrap: wrap;
    }

    .item {
        flex: 1 1 calc(50% - 1rem);
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .item {
        flex: 1 1 calc(33.333% - 1rem);
    }
}

/* Responsive navigation */
.nav {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

@media (max-width: 768px) {
    .nav {
        flex-direction: column;
    }

    .nav-item {
        width: 100%;
        text-align: center;
    }
}
