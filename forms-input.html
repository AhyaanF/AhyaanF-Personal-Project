Forms and Input Elements

Basic Form Structure

HTML forms are used to collect user input. The form element is a container for different types of input elements.

Form Basic Structure


<form action="/submit" method="POST">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>
    
    <button type="submit">Submit</button>
</form>

Input Types

Text Input Types


<!-- Text input -->
<input type="text" placeholder="Enter your name">

<!-- Email input -->
<input type="email" placeholder="Enter your email">

<!-- Password input -->
<input type="password" placeholder="Enter your password">

<!-- Number input -->
<input type="number" min="0" max="100">

<!-- Text area -->
<textarea rows="4" cols="50">Enter your message here...</textarea>

Selection Inputs


<!-- Checkbox -->
<input type="checkbox" id="subscribe" name="subscribe">
<label for="subscribe">Subscribe to newsletter</label>

<!-- Radio buttons -->
<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>
<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

<!-- Dropdown select -->
<select name="country">
    <option value="">Select a country</option>
    <option value="us">United States</option>
    <option value="uk">United Kingdom</option>
    <option value="ca">Canada</option>
</select>

Special Input Types


<!-- Date input -->
<input type="date">

<!-- Color picker -->
<input type="color">

<!-- File upload -->
<input type="file">

<!-- Range slider -->
<input type="range" min="0" max="100" value="50">

Practice Exercise

Create a registration form that includes:

  • Full name (text input)
  • Email address (email input)
  • Password with confirmation
  • Date of birth (date input)
  • Gender selection (radio buttons)
  • Interests (checkboxes)
  • Country selection (dropdown)

<form action="/register" method="POST">
    <div class="form-group">
        <label for="fullname">Full Name:</label>
        <input type="text" id="fullname" name="fullname" required>
    </div>

    <div class="form-group">
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
    </div>

    <div class="form-group">
        <label for="password">Password:</label>
        <input type="password" id="password" name="password" required>
    </div>

    <div class="form-group">
        <label for="dob">Date of Birth:</label>
        <input type="date" id="dob" name="dob" required>
    </div>

    <div class="form-group">
        <label>Gender:</label>
        <input type="radio" id="male" name="gender" value="male">
        <label for="male">Male</label>
        <input type="radio" id="female" name="gender" value="female">
        <label for="female">Female</label>
    </div>

    <div class="form-group">
        <label>Interests:</label>
        <input type="checkbox" id="coding" name="interests[]" value="coding">
        <label for="coding">Coding</label>
        <input type="checkbox" id="design" name="interests[]" value="design">
        <label for="design">Design</label>
    </div>

    <div class="form-group">
        <label for="country">Country:</label>
        <select id="country" name="country" required>
            <option value="">Select a country</option>
            <option value="us">United States</option>
            <option value="uk">United Kingdom</option>
            <option value="ca">Canada</option>
        </select>
    </div>

    <button type="submit">Register</button>
</form>
