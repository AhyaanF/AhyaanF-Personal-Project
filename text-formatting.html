Text Formatting in HTML

Basic Text Formatting

HTML provides various elements to format text and make it more readable and meaningful.

Text Emphasis Elements


<!-- Bold and Strong Text -->
<b>Bold text</b>
<strong>Important text</strong>

<!-- Italic and Emphasized Text -->
<i>Italic text</i>
<em>Emphasized text</em>

<!-- Underlined Text -->
<u>Underlined text</u>

Special Text Formatting


<!-- Marked/Highlighted Text -->
<mark>Highlighted text</mark>

<!-- Subscript and Superscript -->
H<sub>2</sub>O
X<sup>2</sup>

<!-- Strikethrough Text -->
<del>Deleted text</del>
<s>Strikethrough text</s>

<!-- Small Text -->
<small>Smaller text</small>

Text Organization

Line Breaks and Horizontal Rules


This is line one<br>
This is line two

<hr> <!-- Creates a horizontal line -->

Preserving Text Formatting


<pre>
    This text preserves
        both spaces
            and line breaks
</pre>

Practice Exercise

Create a paragraph that includes:

  • A highlighted word
  • A mathematical expression using subscript and superscript
  • Some deleted text
