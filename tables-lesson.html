HTML Tables & Data Structure

Creating Tables

Tables are used to organize data into rows and columns:


<table border="1">
    <!-- Table Header -->
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Country</th>
        </tr>
    </thead>

    <!-- Table Body -->
    <tbody>
        <tr>
            <td>John Doe</td>
            <td>25</td>
            <td>USA</td>
        </tr>
        <tr>
            <td>Jane Smith</td>
            <td>30</td>
            <td>Canada</td>
        </tr>
    </tbody>

    <!-- Table Footer -->
    <tfoot>
        <tr>
            <td colspan="3">Total Entries: 2</td>
        </tr>
    </tfoot>
</table>

<!-- Complex Table with Spanning -->
<table border="1">
    <tr>
        <td rowspan="2">Region</td>
        <td colspan="2">Sales</td>
    </tr>
    <tr>
        <td>2022</td>
        <td>2023</td>
    </tr>
</table>

Table Elements:

  • <table> - Defines a table
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell
  • <thead> - Groups header content
  • <tbody> - Groups body content
  • <tfoot> - Groups footer content

Practice Exercise

Create a table for a class schedule that includes:

  • Headers for Time, Monday through Friday
  • At least three time slots
  • Use rowspan or colspan where appropriate
  • Include a footer with total hours
