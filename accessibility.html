Web Accessibility (a11y)

ARIA Roles and Labels

ARIA (Accessible Rich Internet Applications) helps make web content more accessible to people with disabilities.

Basic ARIA Attributes


<!-- Using roles -->
<div role="navigation" aria-label="Main menu">
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
    </ul>
</div>

<!-- Using aria-labels -->
<button aria-label="Close dialog" onclick="closeDialog()">×</button>

<!-- Using aria-describedby -->
<input type="text" aria-describedby="password-requirements">
<div id="password-requirements">Password must be at least 8 characters</div>

<!-- Using aria-expanded -->
<button aria-expanded="false" aria-controls="menu-content">
    Toggle Menu
</button>
<div id="menu-content">Menu items here</div>

Semantic HTML for Accessibility

Proper Heading Structure


<!-- Good heading structure -->
<h1>Main Title</h1>
<section>
    <h2>Section Title</h2>
    <h3>Subsection Title</h3>
</section>

<!-- Form accessibility -->
<form>
    <label for="username">Username:</label>
    <input 
        type="text" 
        id="username" 
        name="username" 
        aria-required="true"
        aria-invalid="false"
    >
    
    <fieldset>
        <legend>Notification Preferences</legend>
        <input type="checkbox" id="email" name="notifications">
        <label for="email">Email notifications</label>
    </fieldset>
</form>

Images and Media Accessibility

Alt Text and Captions


<!-- Descriptive alt text for images -->
<img 
    src="chart.png" 
    alt="Bar chart showing sales growth from 2020 to 2024"
>

<!-- Decorative images -->
<img src="decoration.png" alt="" role="presentation">

<!-- Figure with caption -->
<figure>
    <img 
        src="diagram.png" 
        alt="System architecture diagram"
    >
    <figcaption>Figure 1: System Architecture Overview</figcaption>
</figure>

<!-- Accessible video player -->
<video controls>
    <source src="video.mp4" type="video/mp4">
    <track 
        kind="captions" 
        src="captions.vtt" 
        srclang="en" 
        label="English"
    >
    <track 
        kind="descriptions" 
        src="descriptions.vtt" 
        srclang="en" 
        label="English descriptions"
    >
</video>

Keyboard Navigation

Focus Management


<!-- Proper tab order -->
<div class="navigation">
    <a href="#main" class="skip-link">Skip to main content</a>
    <nav>
        <a href="#" tabindex="1">Home</a>
        <a href="#" tabindex="2">About</a>
        <a href="#" tabindex="3">Contact</a>
    </nav>
</div>

<!-- Interactive elements -->
<button 
    onclick="toggleMenu()" 
    onkeydown="handleKeyPress(event)"
    aria-expanded="false"
>
    Menu
</button>

<div 
    role="dialog" 
    aria-labelledby="dialog-title"
    aria-modal="true"
>
    <h2 id="dialog-title">Dialog Title</h2>
    <button autofocus>Primary Action</button>
</div>

Practice Exercise

Create an accessible form with the following requirements:

  • Proper form labels and structure
  • Error messaging using ARIA
  • Required field indicators
  • Keyboard navigation support

<form role="form" aria-labelledby="form-title">
    <h2 id="form-title">Contact Form</h2>
    
    <div class="form-group">
        <label for="name">
            Name
            <span class="required" aria-hidden="true">*</span>
        </label>
        <input 
            type="text" 
            id="name" 
            name="name" 
            required 
            aria-required="true"
            aria-describedby="name-error"
        >
        <div id="name-error" class="error" role="alert" aria-live="polite"></div>
    </div>
    
    <div class="form-group">
        <label for="email">
            Email
            <span class="required" aria-hidden="true">*</span>
        </label>
        <input 
            type="email" 
            id="email" 
            name="email" 
            required 
            aria-required="true"
            aria-describedby="email-error"
        >
        <div id="email-error" class="error" role="alert" aria-live="polite"></div>
    </div>
    
    <fieldset>
        <legend>Preferences</legend>
        <div class="checkbox-group">
            <input type="checkbox" id="newsletter" name="newsletter">
            <label for="newsletter">Subscribe to newsletter</label>
        </div>
    </fieldset>
    
    <button 
        type="submit"
        aria-label="Submit form"
    >
        Submit
    </button>
</form>
