CSS Responsive Images

Basic Responsive Images

Learn how to make images fluid and responsive using CSS properties and techniques.

Fluid Images


/* Basic fluid image */
.responsive-img {
    max-width: 100%;
    height: auto;
}

/* Maintain aspect ratio */
.aspect-ratio-img {
    width: 100%;
    aspect-ratio: 16/9;
    object-fit: cover;
}

/* Background image */
.bg-image {
    background-image: url('image.jpg');
    background-size: cover;
    background-position: center;
    height: 300px;
}

/* Picture element with multiple sources */
picture {
    display: block;
    width: 100%;
}

picture img {
    width: 100%;
    height: auto;
}

Art Direction

Use different image crops and sizes for different screen sizes and device capabilities.

Art Direction Techniques


/* HTML Structure */
<picture>
    <source 
        media="(min-width: 1200px)"
        srcset="large.jpg">
    <source 
        media="(min-width: 800px)"
        srcset="medium.jpg">
    <img 
        src="small.jpg" 
        alt="Responsive image"
        loading="lazy">
</picture>

/* CSS Styling */
.art-directed-img {
    width: 100%;
    height: auto;
}

/* Background image art direction */
.hero {
    width: 100%;
    height: 50vh;
    background-size: cover;
    background-position: center;
}

@media (max-width: 768px) {
    .hero {
        background-image: url('mobile-hero.jpg');
    }
}

@media (min-width: 769px) {
    .hero {
        background-image: url('desktop-hero.jpg');
    }
}

Image Performance

Optimize images for different devices while maintaining quality and performance.

Performance Techniques


/* Lazy loading */
.lazy-img {
    loading="lazy";
    width: 100%;
    height: auto;
}

/* Blur-up technique */
.blur-up {
    filter: blur(5px);
    transition: filter 0.3s;
}

.blur-up.loaded {
    filter: blur(0);
}

/* Responsive image set */
<img 
    srcset="small.jpg 300w,
            medium.jpg 600w,
            large.jpg 900w"
    sizes="(max-width: 320px) 280px,
           (max-width: 640px) 580px,
           880px"
    src="fallback.jpg"
    alt="Responsive image">

/* Modern image formats */
<picture>
    <source 
        type="image/webp" 
        srcset="image.webp">
    <source 
        type="image/jpeg" 
        srcset="image.jpg">
    <img 
        src="fallback.jpg" 
        alt="Modern format image">
</picture>

Advanced Techniques

Advanced Image Handling


/* Image grid with aspect ratio */
.image-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}

.grid-item {
    position: relative;
    padding-bottom: 75%; /* 4:3 aspect ratio */
}

.grid-item img {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    object-fit: cover;
}

/* Responsive background patterns */
.pattern-bg {
    background-image: url('pattern.png');
    background-size: 20px 20px;
}

@media (min-resolution: 2dppx) {
    .pattern-bg {
        background-size: 10px 10px;
    }
}

/* Image hover effects */
.hover-zoom {
    overflow: hidden;
}

.hover-zoom img {
    transition: transform 0.3s ease;
}

.hover-zoom:hover img {
    transform: scale(1.1);
}

Practice Exercise

Create a responsive image gallery that:

  • Uses modern image formats with fallbacks
  • Implements lazy loading
  • Has different image sizes for different screens
  • Includes hover effects and transitions

<!-- index.html -->
<div class="gallery">
    <div class="gallery-item">
        <picture>
            <source 
                type="image/webp"
                srcset="large.webp 1200w,
                        medium.webp 800w,
                        small.webp 400w"
                sizes="(max-width: 768px) 100vw,
                       (max-width: 1200px) 50vw,
                       33vw">
            <img 
                src="fallback.jpg"
                alt="Gallery image"
                loading="lazy"
                class="gallery-img">
        </picture>
        <div class="gallery-overlay">
            <h3>Image Title</h3>
            <p>Image description</p>
        </div>
    </div>
</div>

/* styles.css */
.gallery {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: clamp(1rem, 2vw, 2rem);
    padding: min(5vw, 3rem);
}

.gallery-item {
    position: relative;
    overflow: hidden;
    border-radius: 0.5rem;
    aspect-ratio: 16/9;
}

.gallery-img {
    width: 100%;
    height: 100%;
    object-fit: cover;
    transition: transform 0.3s ease;
}

.gallery-overlay {
    position: absolute;
    bottom: 0;
    left: 0;
    right: 0;
    background: linear-gradient(transparent, rgba(0,0,0,0.8));
    color: white;
    padding: 1rem;
    transform: translateY(100%);
    transition: transform 0.3s ease;
}

.gallery-item:hover .gallery-img {
    transform: scale(1.1);
}

.gallery-item:hover .gallery-overlay {
    transform: translateY(0);
}

.gallery-overlay h3 {
    font-size: calc(1rem + 0.5vw);
    margin: 0 0 0.5rem;
}

.gallery-overlay p {
    font-size: calc(0.875rem + 0.25vw);
    margin: 0;
    opacity: 0.9;
}

@media (max-width: 768px) {
    .gallery {
        grid-template-columns: 1fr;
    }
    
    .gallery-overlay {
        transform: translateY(0);
        background: rgba(0,0,0,0.6);
    }
}
