Practice Exercise

Create a color grade that demonstrates:

  • Basic color correction
  • Creative look development
  • Secondary color adjustments
  • Consistent style across scenes

Example Color Grade:

1. Basic Correction
   - Balance white point to 6500K
   - Set black point at 0 IRE
   - Adjust mid-tones to 50 IRE
   - Normalize saturation to 80%

2. Look Development
   - Cool shadows (-5 blue)
   - Warm highlights (+3 orange)
   - Contrast curve (20% stronger)
   - Slight desaturation (-10%)

3. Secondary Work
   - Skin tone isolation (HSL)
   - Sky enhancement (Power Window)
   - Foliage adjustment (Curves)
   - Subtle vignette (20% darker)

4. Scene Matching
   - Shot comparison
   - Waveform matching
   - Color continuity
   - Style consistency
