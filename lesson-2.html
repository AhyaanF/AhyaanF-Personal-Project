Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and more.

Basic CSS Syntax


/* Selector and declaration block */
h1 {
    color: blue;
    font-size: 24px;
    margin-bottom: 20px;
}

/* Class selector */
.highlight {
    background-color: yellow;
    padding: 5px;
}
