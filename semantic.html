Semantic HTML

What is Semantic HTML?

Semantic HTML uses elements that clearly describe their meaning to both the browser and the developer. This improves accessibility, SEO, and code maintainability.

Non-Semantic vs Semantic Elements


<!-- Non-Semantic Elements -->
<div class="header">
    <div class="nav">
        <div class="menu-item">Home</div>
    </div>
</div>

<!-- Semantic Elements -->
<header>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
        </ul>
    </nav>
</header>

Common Semantic Elements

Page Structure Elements


<!-- Basic Page Structure -->
<header>
    <nav>
        <!-- Navigation content -->
    </nav>
</header>

<main>
    <article>
        <section>
            <h2>Article Section</h2>
            <p>Content here...</p>
        </section>
    </article>

    <aside>
        <!-- Sidebar content -->
    </aside>
</main>

<footer>
    <!-- Footer content -->
</footer>

Content Elements


<!-- Article Content -->
<article>
    <header>
        <h1>Article Title</h1>
        <time datetime="2024-02-20">February 20, 2024</time>
    </header>

    <section>
        <h2>Introduction</h2>
        <p>Article content...</p>
    </section>

    <figure>
        <img src="image.jpg" alt="Description">
        <figcaption>Image caption</figcaption>
    </figure>

    <footer>
        <address>
            By <a href="mailto:author@example.com">Author Name</a>
        </address>
    </footer>
</article>

Practice Exercise

Create a semantic HTML structure for a blog post that includes:

  • Proper page structure (header, main, footer)
  • Article with header and sections
  • Navigation menu
  • Sidebar with related content
  • Proper use of time and address elements

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My Blog Post</title>
</head>
<body>
    <header>
        <h1>My Blog</h1>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <article>
            <header>
                <h2>My First Blog Post</h2>
                <time datetime="2024-02-20">
                    Published: February 20, 2024
                </time>
            </header>

            <section>
                <h3>Introduction</h3>
                <p>Welcome to my first blog post...</p>
            </section>

            <section>
                <h3>Main Content</h3>
                <p>The main content goes here...</p>
                
                <figure>
                    <img src="blog-image.jpg" 
                         alt="Related to blog post">
                    <figcaption>
                        Image description
                    </figcaption>
                </figure>
            </section>
        </article>

        <aside>
            <h3>Related Posts</h3>
            <ul>
                <li><a href="#">Another Post</a></li>
                <li><a href="#">More Content</a></li>
            </ul>
        </aside>
    </main>

    <footer>
        <address>
            Written by <a href="mailto:author@example.com">
            Author Name</a>
        </address>
        <p>© 2024 My Blog. All rights reserved.</p>
    </footer>
</body>
</html>
