Responsive Units in CSS

Relative Length Units

Learn about CSS units that scale relative to other elements or viewport dimensions.

Font-Relative Units


/* em - Relative to parent element's font size */
.parent {
    font-size: 16px;
}
.child {
    font-size: 1.5em;    /* 24px (16 × 1.5) */
    padding: 1em;        /* 24px (relative to element's font size) */
    margin: 0.5em;       /* 12px */
}

/* rem - Relative to root element's font size */
:root {
    font-size: 16px;
}
.element {
    font-size: 1.5rem;   /* 24px (16 × 1.5) */
    padding: 1rem;       /* 16px */
    margin: 2rem;        /* 32px */
}

/* ch - Width of the "0" character */
.text-box {
    width: 45ch;         /* Width of 45 "0" characters */
    max-width: 60ch;     /* Maximum width of 60 "0" characters */
}

Viewport-Relative Units


/* vw - Viewport Width */
.hero {
    width: 100vw;        /* Full viewport width */
    height: 50vw;        /* Half viewport width */
}

/* vh - Viewport Height */
.full-screen {
    height: 100vh;       /* Full viewport height */
    min-height: 50vh;    /* Minimum half viewport height */
}

/* vmin & vmax */
.responsive-square {
    width: 50vmin;       /* 50% of viewport's smaller dimension */
    height: 50vmin;      /* Maintains square shape */
}

.dynamic-size {
    font-size: 5vw;      /* Text scales with viewport width */
    padding: 2vh 2vw;    /* Padding scales with viewport */
}

Percentage Units

Percentage-Based Layouts


/* Basic percentage layout */
.container {
    width: 80%;          /* 80% of parent width */
    max-width: 1200px;   /* Maximum width constraint */
    margin: 0 auto;      /* Center horizontally */
}

.sidebar {
    width: 30%;          /* 30% of parent width */
    float: left;
}

.main-content {
    width: 70%;          /* 70% of parent width */
    float: right;
}

/* Percentage in padding and margin */
.box {
    padding: 10%;        /* Padding relative to parent width */
    margin-bottom: 5%;   /* Margin relative to parent width */
}

/* Aspect ratio using padding percentage */
.responsive-video {
    position: relative;
    width: 100%;
    padding-bottom: 56.25%; /* 16:9 aspect ratio */
}

.responsive-video iframe {
    position: absolute;
    width: 100%;
    height: 100%;
    top: 0;
    left: 0;
}

Fluid Typography

Responsive Text Sizing


/* Basic fluid typography */
html {
    font-size: 16px;
}

@media screen and (min-width: 320px) {
    html {
        font-size: calc(16px + 2vw);
    }
}

@media screen and (min-width: 1200px) {
    html {
        font-size: 24px;
    }
}

/* Clamp for fluid typography */
.heading {
    font-size: clamp(1.5rem, 5vw, 3rem);
    /* Minimum: 1.5rem
       Preferred: 5vw
       Maximum: 3rem */
}

/* Fluid spacing */
.container {
    padding: clamp(1rem, 5vw, 3rem);
    gap: clamp(1rem, 2vw, 2rem);
}

Practice Exercise

Create a responsive layout using various CSS units:

  • Fluid typography using clamp()
  • Responsive container with percentage and max-width
  • Viewport-based hero section
  • Responsive card with em/rem units

/* HTML Structure */
<div class="container">
    <header class="hero">
        <h1 class="hero-title">Welcome</h1>
    </header>
    
    <main class="content">
        <div class="card">
            <h2>Card Title</h2>
            <p>Card content goes here</p>
        </div>
    </main>
</div>

/* CSS Styles */
:root {
    font-size: 16px;
}

/* Container */
.container {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
    padding: clamp(1rem, 5vw, 3rem);
}

/* Hero Section */
.hero {
    height: 80vh;
    min-height: 400px;
    display: flex;
    align-items: center;
    justify-content: center;
}

.hero-title {
    font-size: clamp(2rem, 8vw, 5rem);
    text-align: center;
}

/* Content Area */
.content {
    margin: 2rem auto;
    display: grid;
    gap: clamp(1rem, 3vw, 2rem);
}

/* Card */
.card {
    padding: 1.5rem;
    border-radius: 0.5rem;
    box-shadow: 0 0.25rem 1rem rgba(0, 0, 0, 0.1);
}

.card h2 {
    font-size: 1.5rem;
    margin-bottom: 1rem;
}

.card p {
    font-size: 1rem;
    line-height: 1.6;
}

/* Responsive Adjustments */
@media (min-width: 768px) {
    .content {
        grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    }
}
