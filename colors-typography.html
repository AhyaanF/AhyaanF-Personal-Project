Colors and Typography in CSS

Working with Colors

CSS provides multiple ways to specify colors, each with its own advantages.

Color Values


/* Named Colors */
.named-colors {
    color: red;
    background-color: lightblue;
    border-color: navy;
}

/* Hexadecimal Colors */
.hex-colors {
    color: #ff0000;        /* Red */
    background: #00ff00;   /* Green */
    border: #0000ff;      /* Blue */
}

/* RGB and RGBA Colors */
.rgb-colors {
    color: rgb(255, 0, 0);             /* Red */
    background: rgba(0, 255, 0, 0.5);  /* Semi-transparent green */
}

/* HSL and HSLA Colors */
.hsl-colors {
    color: hsl(0, 100%, 50%);           /* Red */
    background: hsla(120, 100%, 50%, 0.5); /* Semi-transparent green */
}

/* Modern Color Functions */
.modern-colors {
    color: color-mix(in srgb, #ff0000 50%, #00ff00 50%);
    background: color-contrast(#ffffff vs #000000, #444444, #666666);
}

Typography Basics

Font Properties


/* Font Family */
body {
    font-family: Arial, Helvetica, sans-serif;  /* Fallback system */
}

h1 {
    font-family: 'Times New Roman', Times, serif;
}

/* Font Size */
.font-sizes {
    font-size: 16px;          /* Pixels */
    font-size: 1.2em;         /* Relative to parent */
    font-size: 1.2rem;        /* Relative to root */
    font-size: 120%;          /* Percentage */
}

/* Font Weight */
.font-weights {
    font-weight: normal;      /* or 400 */
    font-weight: bold;        /* or 700 */
    font-weight: 300;         /* Light */
    font-weight: 600;         /* Semi-bold */
}

/* Font Style */
.font-styles {
    font-style: normal;
    font-style: italic;
    font-style: oblique;
}

Text Properties


/* Text Alignment */
.text-alignment {
    text-align: left;
    text-align: right;
    text-align: center;
    text-align: justify;
}

/* Line Height */
.line-spacing {
    line-height: 1.6;        /* Unitless (recommended) */
    line-height: 24px;       /* Fixed height */
    line-height: 150%;       /* Percentage */
}

/* Text Decoration */
.text-decoration {
    text-decoration: none;
    text-decoration: underline;
    text-decoration: line-through;
    text-decoration: overline;
}

/* Text Transform */
.text-transform {
    text-transform: uppercase;
    text-transform: lowercase;
    text-transform: capitalize;
}

/* Letter and Word Spacing */
.spacing {
    letter-spacing: 2px;
    word-spacing: 4px;
}

Web Fonts

Using Google Fonts


<!-- In HTML head -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

/* In CSS */
body {
    font-family: 'Roboto', sans-serif;
}

Using @font-face


@font-face {
    font-family: 'CustomFont';
    src: url('path/to/font.woff2') format('woff2'),
         url('path/to/font.woff') format('woff');
    font-weight: normal;
    font-style: normal;
}

.custom-font {
    font-family: 'CustomFont', sans-serif;
}

  • Use different color formats
  • Include custom fonts
  • Apply various text styles
  • Create a consistent typographic hierarchy
