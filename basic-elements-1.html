Basic HTML Elements

Common HTML Elements

HTML provides various elements to structure your content. Let's explore the most commonly used ones.

Text Elements


<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph of text.</p>
<br> <!-- Line break -->
<hr> <!-- Horizontal rule -->

Links and Images


<a href="https://example.com">Click here</a>
<img src="image.jpg" alt="Description">

Practice Exercise

Create a simple webpage that includes:

  • A main heading
  • Two paragraphs of text
  • An image
  • A link to your favorite website

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My Practice Page</title>
</head>
<body>
    <h1>Welcome to My Page</h1>
    
    <p>This is my first paragraph. I'm learning HTML!</p>
    <p>This is my second paragraph. Web development is fun!</p>
    
    <img src="cat.jpg" alt="A cute cat">
    
    <p><a href="https://www.example.com">Visit my favorite website</a></p>
</body>
</html>
