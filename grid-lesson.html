CSS Grid Layout

Understanding CSS Grid

CSS Grid is a two-dimensional layout system that lets you create complex web layouts with rows and columns.

Grid Container Properties


/* Grid Container */
.container {
    display: grid;
    /* or display: inline-grid */
    
    /* Define columns */
    grid-template-columns: 200px 200px 200px;
    grid-template-columns: repeat(3, 1fr);
    grid-template-columns: minmax(100px, 1fr) 2fr 1fr;
    
    /* Define rows */
    grid-template-rows: 100px auto 100px;
    grid-template-rows: repeat(3, minmax(100px, auto));
    
    /* Grid gaps */
    gap: 20px;
    column-gap: 20px;
    row-gap: 20px;
    
    /* Grid areas */
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

Grid Item Properties


/* Grid Items */
.item {
    /* Grid lines positioning */
    grid-column: 1 / 3;
    grid-column: span 2;
    grid-row: 1 / 3;
    grid-row: span 2;
    
    /* Shorthand */
    grid-area: 1 / 1 / 3 / 3;
    
    /* Named grid areas */
    grid-area: header;
    
    /* Alignment */
    justify-self: start | end | center | stretch;
    align-self: start | end | center | stretch;
}

Advanced Grid Features

Grid Alignment and Auto Features


.container {
    /* Container alignment */
    justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
    align-content: start | end | center | stretch | space-around | space-between | space-evenly;
    
    /* Items alignment */
    justify-items: start | end | center | stretch;
    align-items: start | end | center | stretch;
    
    /* Auto features */
    grid-auto-columns: 100px;
    grid-auto-rows: minmax(100px, auto);
    grid-auto-flow: row | column | dense;
}

/* Responsive grid */
@media (max-width: 768px) {
    .container {
        grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    }
}

Practice Exercise

Create a responsive webpage layout using CSS Grid that includes:

  • A header spanning the full width
  • A main content area with sidebar
  • A footer spanning the full width
  • Responsive behavior on mobile devices

/* HTML Structure */
<div class="grid-container">
    <header class="header">Header</header>
    <nav class="sidebar">Sidebar</nav>
    <main class="main-content">Main Content</main>
    <footer class="footer">Footer</footer>
</div>

/* CSS Styles */
.grid-container {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-template-columns: 200px 1fr 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 20px;
    min-height: 100vh;
}

.header {
    grid-area: header;
    background: #333;
    color: white;
    padding: 1rem;
}

.sidebar {
    grid-area: sidebar;
    background: #f4f4f4;
    padding: 1rem;
}

.main-content {
    grid-area: main;
    padding: 1rem;
}

.footer {
    grid-area: footer;
    background: #333;
    color: white;
    padding: 1rem;
}

/* Responsive Design */
@media (max-width: 768px) {
    .grid-container {
        grid-template-areas:
            "header"
            "main"
            "sidebar"
            "footer";
        grid-template-columns: 1fr;
    }
}
