Modern HTML5 Features

New HTML5 Elements and APIs

HTML5 introduces powerful new features and elements:


<!-- New Semantic Elements -->
<header>
    <nav>
        <menu>
            <menuitem>Home</menuitem>
        </menu>
    </nav>
</header>

<!-- Canvas for Graphics -->
<canvas id="myCanvas" width="200" height="100">
    Your browser does not support canvas
</canvas>

<!-- Audio and Video -->
<video controls preload="auto">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <track src="subtitles.vtt" kind="subtitles" srclang="en">
</video>

<!-- Form Improvements -->
<form>
    <input type="email" required>
    <input type="date">
    <input type="range" min="0" max="100">
    <input type="search" placeholder="Search...">
    <input type="color">
    <datalist id="browsers">
        <option value="Chrome">
        <option value="Firefox">
    </datalist>
</form>

<!-- Local Storage -->
<script>
localStorage.setItem('username', 'John');
sessionStorage.setItem('temp', 'data');
</script>

<!-- Drag and Drop -->
<div draggable="true" ondragstart="drag(event)">
    Drag me!
</div>
<div ondrop="drop(event)" ondragover="allowDrop(event)">
    Drop here
</div>

<!-- Geolocation -->
<button onclick="getLocation()">Get Location</button>

<!-- Web Workers -->
<script>
const worker = new Worker('worker.js');
worker.postMessage('Start working');
</script>

HTML5 Features:

  • Semantic Elements - Better structure
  • Canvas & SVG - Graphics and animations
  • Media Elements - Native audio and video
  • Form Improvements - New input types
  • Web Storage - Local and session storage
  • Drag & Drop - Native drag and drop
  • Geolocation - Location services
  • Web Workers - Background processing

Practice Exercise

  • A canvas element with basic drawing
  • A video player with controls
  • New HTML5 form elements
  • Local storage usage
  • Drag and drop functionality
