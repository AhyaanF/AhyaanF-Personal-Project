CSS Filters

Basic Filters

Learn how to apply visual effects to elements using CSS filters.

Common Filter Functions


/* Blur effect */
.blur {
    filter: blur(5px);
}

/* Brightness adjustment */
.brightness {
    filter: brightness(150%);
}

/* Contrast adjustment */
.contrast {
    filter: contrast(200%);
}

/* Grayscale conversion */
.grayscale {
    filter: grayscale(100%);
}

/* Hue rotation */
.hue-rotate {
    filter: hue-rotate(90deg);
}

/* Invert colors */
.invert {
    filter: invert(100%);
}

/* Opacity adjustment */
.opacity {
    filter: opacity(50%);
}

/* Saturation adjustment */
.saturate {
    filter: saturate(200%);
}

/* Sepia effect */
.sepia {
    filter: sepia(100%);
}

Multiple Filters


/* Combining multiple filters */
.multi-filter {
    filter: 
        contrast(150%) 
        brightness(120%) 
        blur(2px);
}

/* Image enhancement */
.enhance-image {
    filter: 
        brightness(110%) 
        contrast(110%) 
        saturate(120%);
}

/* Vintage effect */
.vintage {
    filter: 
        sepia(50%) 
        contrast(150%) 
        brightness(90%) 
        hue-rotate(-30deg);
}

/* Dramatic effect */
.dramatic {
    filter: 
        contrast(150%) 
        brightness(90%) 
        grayscale(50%);
}

Advanced Filter Techniques

Filter Transitions


/* Hover effects with filters */
.hover-filter {
    filter: grayscale(100%);
    transition: filter 0.3s ease;
}

.hover-filter:hover {
    filter: grayscale(0%);
}

/* Multiple filter transition */
.complex-hover {
    filter: 
        grayscale(100%) 
        blur(0px);
    transition: filter 0.3s ease;
}

.complex-hover:hover {
    filter: 
        grayscale(0%) 
        blur(2px);
}

/* Animated filters */
@keyframes filter-animation {
    0% {
        filter: hue-rotate(0deg);
    }
    100% {
        filter: hue-rotate(360deg);
    }
}

.animated-filter {
    animation: filter-animation 5s infinite linear;
}

Backdrop Filters


/* Basic backdrop filter */
.backdrop {
    backdrop-filter: blur(10px);
    background-color: rgba(255, 255, 255, 0.5);
}

/* Glass effect */
.glass {
    background: rgba(255, 255, 255, 0.2);
    backdrop-filter: 
        blur(10px) 
        saturate(180%);
}

/* Dark overlay */
.dark-overlay {
    background: rgba(0, 0, 0, 0.5);
    backdrop-filter: 
        blur(5px) 
        brightness(70%);
}

/* Frosted glass */
.frosted {
    background: rgba(255, 255, 255, 0.1);
    backdrop-filter: 
        blur(10px) 
        contrast(90%) 
        brightness(110%);
}

Practice Exercise

Create an image gallery with filter effects that includes:

  • Hover transitions with filters
  • Different filter combinations
  • Backdrop filter overlays
  • Animated filter effects

/* HTML Structure */
<div class="gallery">
    <div class="gallery-item">
        <img src="image1.jpg" alt="Gallery Image 1">
        <div class="overlay">
            <h3>Image Title</h3>
            <p>Image description</p>
        </div>
    </div>
    <!-- More gallery items... -->
</div>

/* CSS Styles */
.gallery {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
    padding: 20px;
}

.gallery-item {
    position: relative;
    overflow: hidden;
    border-radius: 10px;
    cursor: pointer;
}

.gallery-item img {
    width: 100%;
    height: 300px;
    object-fit: cover;
    filter: grayscale(100%) brightness(90%);
    transition: filter 0.3s ease,
                transform 0.3s ease;
}

.gallery-item:hover img {
    filter: grayscale(0%) brightness(100%);
    transform: scale(1.05);
}

.overlay {
    position: absolute;
    inset: 0;
    background: rgba(0, 0, 0, 0.3);
    backdrop-filter: blur(5px);
    color: white;
    padding: 20px;
    opacity: 0;
    transition: opacity 0.3s ease;
    display: flex;
    flex-direction: column;
    justify-content: flex-end;
}

.gallery-item:hover .overlay {
    opacity: 1;
}

/* Filter Variations */
.vintage img {
    filter: 
        sepia(50%) 
        contrast(150%) 
        brightness(90%) 
        hue-rotate(-30deg);
}

.dramatic img {
    filter: 
        contrast(150%) 
        brightness(90%) 
        grayscale(50%);
}

.vibrant img {
    filter: 
        brightness(110%) 
        contrast(110%) 
        saturate(150%);
}

/* Animated Filters */
@keyframes hue-rotation {
    0% {
        filter: hue-rotate(0deg);
    }
    100% {
        filter: hue-rotate(360deg);
    }
}

.animated img {
    animation: hue-rotation 10s infinite linear;
}

/* Responsive Design */
@media (max-width: 768px) {
    .gallery {
        grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    }
    
    .gallery-item img {
        height: 250px;
    }
}

/* Performance Optimization */
.gallery-item img {
    will-change: filter, transform;
}

@media (prefers-reduced-motion: reduce) {
    .gallery-item img,
    .overlay {
        transition: none;
    }
    
    .animated img {
        animation: none;
    }
}
