HTML Attributes

Understanding HTML Attributes

Attributes provide additional information about HTML elements and modify their behavior:


<!-- Link Attributes -->
<a href="https://example.com" target="_blank" title="Visit Example">
    Click here
</a>

<!-- Image Attributes -->
<img src="photo.jpg" alt="A beautiful sunset" width="300" height="200">

<!-- Input Attributes -->
<input type="text" placeholder="Enter your name" required>

<!-- Class and ID Attributes -->
<div class="container" id="main-content">
    <p class="text-large">Styled text</p>
</div>

<!-- Data Attributes -->
<button data-action="submit" data-id="123">
    Submit Form
</button>

Common Attributes:

  • href - Specifies URL for links
  • src - Specifies source for images and scripts
  • alt - Alternative text for images
  • class - Assigns CSS classes
  • id - Unique identifier for elements
  • style - Inline CSS styles
  • title - Tooltip text
