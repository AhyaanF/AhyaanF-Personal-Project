Flexbox Layout

Understanding Flexbox

Flexbox is a one-dimensional layout model that helps distribute space and align content in a container.

Basic Flexbox Container


/* Flex Container Properties */
.container {
    display: flex;
    /* or display: inline-flex */
    
    /* Main Axis */
    flex-direction: row;         /* default */
    flex-direction: row-reverse;
    flex-direction: column;
    flex-direction: column-reverse;
    
    /* Main Axis Alignment */
    justify-content: flex-start;  /* default */
    justify-content: flex-end;
    justify-content: center;
    justify-content: space-between;
    justify-content: space-around;
    justify-content: space-evenly;
    
    /* Cross Axis Alignment */
    align-items: stretch;        /* default */
    align-items: flex-start;
    align-items: flex-end;
    align-items: center;
    align-items: baseline;
}

Flex Items Properties


/* Flex Item Properties */
.item {
    /* Growth & Shrink */
    flex-grow: 0;      /* default */
    flex-shrink: 1;    /* default */
    flex-basis: auto;  /* default */
    
    /* Shorthand */
    flex: 0 1 auto;    /* default */
    flex: 1;           /* 1 1 0% */
    flex: auto;        /* 1 1 auto */
    flex: none;        /* 0 0 auto */
    
    /* Individual Alignment */
    align-self: auto;      /* default */
    align-self: flex-start;
    align-self: flex-end;
    align-self: center;
    align-self: baseline;
    align-self: stretch;
    
    /* Order */
    order: 0;          /* default */
}

Advanced Flexbox

Multi-line Flex Containers


.container {
    /* Wrapping */
    flex-wrap: nowrap;     /* default */
    flex-wrap: wrap;
    flex-wrap: wrap-reverse;
    
    /* Multi-line Alignment */
    align-content: flex-start;
    align-content: flex-end;
    align-content: center;
    align-content: space-between;
    align-content: space-around;
    align-content: stretch;    /* default */
}

/* Common Patterns */
.equal-width {
    flex: 1;
}

.fixed-width {
    flex: 0 0 200px;
}

.responsive-item {
    flex: 1 1 300px;
}

Practice Exercise

Create a responsive layout that includes:

  • A navigation bar with evenly spaced items
  • A card layout that wraps on smaller screens
  • A footer with centered content

/* HTML Structure */
<nav class="navbar">
    <div class="nav-items">
        <a href="#">Home</a>
        <a href="#">About</a>
        <a href="#">Services</a>
        <a href="#">Contact</a>
    </div>
</nav>

<div class="card-container">
    <div class="card">Card 1</div>
    <div class="card">Card 2</div>
    <div class="card">Card 3</div>
</div>

<footer class="footer">
    <p>© 2024 Your Website</p>
</footer>

/* CSS Styles */
.navbar {
    background: #333;
    padding: 1rem;
}

.nav-items {
    display: flex;
    justify-content: space-evenly;
    max-width: 800px;
    margin: 0 auto;
}

.nav-items a {
    color: white;
    text-decoration: none;
}

.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
    padding: 20px;
}

.card {
    flex: 1 1 300px;
    padding: 20px;
    border: 1px solid #ddd;
    border-radius: 8px;
}

.footer {
    display: flex;
    justify-content: center;
    align-items: center;
    padding: 20px;
    background: #f5f5f5;
}
