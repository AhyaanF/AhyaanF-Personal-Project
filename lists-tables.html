Lists and Tables in HTML

HTML Lists

HTML provides three types of lists to organize content: unordered, ordered, and description lists.

Unordered Lists


<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Ordered Lists


<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

<!-- With different numbering -->
<ol type="A"> <!-- A, B, C -->
    <li>Item A</li>
    <li>Item B</li>
</ol>

Description Lists


<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

HTML Tables

Tables are used to display data in rows and columns.

Basic Table Structure


<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
        </tr>
    </tfoot>
</table>

Table with Colspan and Rowspan


<table>
    <tr>
        <th colspan="2">Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>Doe</td>
        <td rowspan="2">25</td>
    </tr>
</table>

  • An unordered list of ingredients for one of those foods
  • A table showing a weekly schedule with merged cells
