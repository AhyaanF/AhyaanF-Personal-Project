Responsive Web Design

Media Queries

Media queries allow you to apply CSS rules based on device characteristics like screen size, orientation, and resolution.

Basic Media Queries


/* Mobile First Approach */
/* Base styles for mobile */
.container {
    width: 100%;
    padding: 15px;
}

/* Tablet styles */
@media screen and (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* Desktop styles */
@media screen and (min-width: 1024px) {
    .container {
        width: 960px;
    }
}

/* Large desktop styles */
@media screen and (min-width: 1200px) {
    .container {
        width: 1140px;
    }
}

/* Device orientation */
@media (orientation: landscape) {
    .sidebar {
        width: 30%;
        float: left;
    }
}

/* Print styles */
@media print {
    .no-print {
        display: none;
    }
}

Complex Media Queries


/* Combining conditions */
@media screen and (min-width: 768px) and (max-width: 1023px) {
    /* Tablet-specific styles */
}

/* High-resolution displays */
@media screen and (min-resolution: 192dpi),
       screen and (-webkit-min-device-pixel-ratio: 2) {
    /* High-DPI styles */
}

/* Feature queries */
@supports (display: grid) {
    .container {
        display: grid;
        grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    }
}

Responsive Layout Patterns

Common Responsive Patterns


/* Mostly Fluid Pattern */
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
    padding: 20px;
}

/* Column Drop Pattern */
.container {
    display: flex;
    flex-direction: column;
}

@media (min-width: 768px) {
    .container {
        flex-direction: row;
    }
    
    .sidebar {
        flex: 1;
    }
    
    .main-content {
        flex: 2;
    }
}

/* Layout Shifter Pattern */
@media (min-width: 768px) {
    .container {
        display: grid;
        grid-template-areas:
            "header header"
            "nav main"
            "footer footer";
    }
}

Responsive Images

Responsive Image Techniques


/* Basic responsive image */
img {
    max-width: 100%;
    height: auto;
}

/* Art direction with picture element */
<picture>
    <source media="(min-width: 800px)" srcset="hero-large.jpg">
    <source media="(min-width: 400px)" srcset="hero-medium.jpg">
    <img src="hero-small.jpg" alt="Hero image">
</picture>

/* Responsive background images */
.hero {
    background-image: url('small.jpg');
    background-size: cover;
}

@media (min-width: 768px) {
    .hero {
        background-image: url('large.jpg');
    }
}

Practice Exercise

Create a responsive webpage that includes:

  • A responsive navigation menu (mobile hamburger menu)
  • A fluid grid of cards
  • Responsive images with art direction
  • Different layouts for mobile and desktop

/* HTML Structure */
<header class="site-header">
    <nav class="nav-container">
        <button class="menu-toggle">☰</button>
        <ul class="nav-menu">
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
            <li><a href="#">Services</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>
</header>

<main class="content">
    <picture class="hero-image">
        <source media="(min-width: 1024px)" srcset="hero-large.jpg">
        <source media="(min-width: 768px)" srcset="hero-medium.jpg">
        <img src="hero-small.jpg" alt="Hero image">
    </picture>
    
    <div class="card-grid">
        <div class="card">Card 1</div>
        <div class="card">Card 2</div>
        <div class="card">Card 3</div>
    </div>
</main>

/* CSS Styles */
/* Mobile First Styles */
.nav-container {
    padding: 1rem;
}

.menu-toggle {
    display: block;
}

.nav-menu {
    display: none;
}

.nav-menu.active {
    display: block;
}

.card-grid {
    display: grid;
    grid-template-columns: 1fr;
    gap: 20px;
    padding: 20px;
}

/* Tablet Styles */
@media (min-width: 768px) {
    .menu-toggle {
        display: none;
    }
    
    .nav-menu {
        display: flex;
        gap: 20px;
    }
    
    .card-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* Desktop Styles */
@media (min-width: 1024px) {
    .nav-container {
        max-width: 1200px;
        margin: 0 auto;
    }
    
    .card-grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

/* JavaScript for mobile menu */
document.querySelector('.menu-toggle').addEventListener('click', function() {
    document.querySelector('.nav-menu').classList.toggle('active');
});
