CSS Positioning

Position Property

The position property specifies how an element is positioned in a document.

Position Values


/* Static positioning (default) */
.static {
    position: static;
    /* Offset properties have no effect */
}

/* Relative positioning */
.relative {
    position: relative;
    top: 20px;
    left: 20px;
    /* Offset from normal position */
}

/* Absolute positioning */
.absolute {
    position: absolute;
    top: 0;
    right: 0;
    /* Positioned relative to nearest positioned ancestor */
}

/* Fixed positioning */
.fixed {
    position: fixed;
    bottom: 20px;
    right: 20px;
    /* Positioned relative to viewport */
}

/* Sticky positioning */
.sticky {
    position: sticky;
    top: 0;
    /* Sticks to viewport when scrolling past threshold */
}

Z-Index and Stacking Context

Z-Index Usage


/* Basic z-index */
.overlay {
    position: fixed;
    z-index: 100;
    /* Higher values appear on top */
}

/* Stacking context example */
.parent {
    position: relative;
    z-index: 1;
}

.child {
    position: absolute;
    z-index: 999;
    /* Only compared with siblings */
}

/* Common z-index values */
:root {
    --z-dropdown: 1000;
    --z-sticky: 1020;
    --z-modal: 1030;
    --z-popover: 1040;
    --z-tooltip: 1050;
}

Offset Properties

Positioning with Offsets


/* Basic offsets */
.element {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    /* Fills entire container */
}

/* Centering with absolute positioning */
.centered {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    /* Perfect centering */
}

/* Responsive offsets */
.responsive-element {
    position: absolute;
    top: 10%;
    left: 10%;
    width: 80%;
    height: 80%;
    /* Percentage-based positioning */
}

/* Negative offsets */
.overlap {
    position: relative;
    left: -20px;
    /* Moves element left */
}

Common Use Cases

Modal Dialog


/* Modal overlay */
.modal-overlay {
    position: fixed;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: rgba(0, 0, 0, 0.5);
    z-index: var(--z-modal);
}

/* Modal content */
.modal-content {
    position: fixed;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    background: white;
    padding: 20px;
    z-index: calc(var(--z-modal) + 1);
}

/* Sticky header */
.header {
    position: sticky;
    top: 0;
    background: white;
    z-index: var(--z-sticky);
}

/* Dropdown menu */
.dropdown {
    position: relative;
}

.dropdown-content {
    position: absolute;
    top: 100%;
    left: 0;
    display: none;
    z-index: var(--z-dropdown);
}

.dropdown:hover .dropdown-content {
    display: block;
}
