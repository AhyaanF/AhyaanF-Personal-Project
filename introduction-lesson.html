Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML.

CSS Syntax


/* Basic CSS Syntax */
selector {
    property: value;
}

/* Example */
h1 {
    color: blue;
    font-size: 24px;
    margin-bottom: 20px;
}

Ways to Include CSS

1. External CSS


/* In HTML file */
<head>
    <link rel="stylesheet" href="styles.css">
</head>

/* In styles.css */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

2. Internal CSS


<head>
    <style>
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
    </style>
</head>

3. Inline CSS


<p style="color: blue; font-size: 16px;">
    This is a paragraph with inline styles
</p>

CSS Selectors

Basic Selectors


/* Element Selector */
p {
    color: black;
}

/* Class Selector */
.highlight {
    background-color: yellow;
}

/* ID Selector */
#header {
    background-color: navy;
}

/* Descendant Selector */
nav a {
    text-decoration: none;
}

/* Child Selector */
ul > li {
    list-style: none;
}

Practice Exercise

Create a webpage that includes:

  • External CSS file
  • Different types of selectors
  • Basic styling properties

/* styles.css */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 20px;
}

.container {
    max-width: 800px;
    margin: 0 auto;
}

#main-header {
    background-color: #333;
    color: white;
    padding: 1rem;
    text-align: center;
}

.nav-menu {
    background-color: #f4f4f4;
    padding: 10px;
}

.nav-menu ul {
    list-style: none;
    padding: 0;
}

.nav-menu li {
    display: inline;
    margin-right: 10px;
}

.nav-menu a {
    color: #333;
    text-decoration: none;
}

.nav-menu a:hover {
    color: #007bff;
}

/* HTML File */
<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <header id="main-header">
            <h1>My Website</h1>
        </header>
        <nav class="nav-menu">
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </div>
</body>
</html>
