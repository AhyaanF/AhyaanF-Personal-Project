Modern CSS Features

Latest CSS Features and Properties

Explore modern CSS capabilities and new features:


/* Container Queries */
.container {
    container-type: inline-size;
    container-name: sidebar;
}

@container sidebar (min-width: 400px) {
    .card {
        display: grid;
        grid-template-columns: 2fr 1fr;
    }
}

/* Logical Properties */
.element {
    margin-block-start: 1em;
    margin-inline-end: 2em;
    padding-inline: 1rem;
    border-block: 1px solid;
}

/* Custom Properties with @property */
@property --gradient-angle {
    syntax: '';
    initial-value: 0deg;
    inherits: false;
}

/* Scroll Snap */
.scroll-container {
    scroll-snap-type: y mandatory;
    overflow-y: scroll;
}

.scroll-item {
    scroll-snap-align: start;
    height: 100vh;
}

/* Aspect Ratio */
.responsive-image {
    aspect-ratio: 16 / 9;
    width: 100%;
    object-fit: cover;
}

/* Backdrop Filter */
.glass-effect {
    backdrop-filter: blur(10px);
    background: rgba(255, 255, 255, 0.1);
}

/* Content Visibility */
.lazy-section {
    content-visibility: auto;
    contain-intrinsic-size: 0 500px;
}

/* Subgrid */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

.nested-grid {
    display: grid;
    grid-template-columns: subgrid;
}

/* :has() Selector */
.card:has(img) {
    padding: 1rem;
}

article:has(> img) {
    margin-bottom: 2rem;
}

/* Color Functions */
.modern-colors {
    color: rgb(from var(--brand-color) r g b);
    background: color-mix(in srgb, #34c9eb 25%, white);
}

/* Layers */
@layer base, components, utilities;

@layer base {
    /* Base styles */
}

/* Parent Selector */
.parent {
    &:hover {
        & > .child {
            color: blue;
        }
    }
}

/* Custom Media Queries */
@custom-media --mobile (width <= 768px);

@media (--mobile) {
    /* Mobile styles */
}

/* Masonry Layout */
.masonry {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    grid-template-rows: masonry;
}

/* Accent Color */
.form-element {
    accent-color: var(--brand-color);
}

Modern CSS Features:

  • Container Queries - Style based on container size
  • Logical Properties - Direction-independent properties
  • Custom Properties - Advanced variable features
  • Modern Selectors - :has() and other new selectors
  • Layout Features - Subgrid, masonry, etc.
  • Performance Features - content-visibility, contain
