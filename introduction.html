Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of HTML documents. It defines how elements should be displayed on screen, on paper, or in other media.

CSS Syntax


/* Basic CSS syntax */
selector {
    property: value;
    another-property: value;
}

/* Example with actual properties */
h1 {
    color: blue;
    font-size: 24px;
    margin-bottom: 20px;
}

Ways to Include CSS

External CSS


<!-- In your HTML file -->
<head>
    <link rel="stylesheet" href="styles.css">
</head>

/* In styles.css */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 20px;
}

Internal CSS


<head>
    <style>
        body {
            font-family: Arial, sans-serif;
            line-height: 1.6;
        }
        
        h1 {
            color: navy;
            text-align: center;
        }
    </style>
</head>

Inline CSS


<p style="color: red; font-size: 16px;">
    This is a paragraph with inline styles
</p>

Selectors and Specificity

Basic Selectors


/* Element selector */
p {
    color: black;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* ID selector */
#header {
    background-color: navy;
    color: white;
}

/* Descendant selector */
article p {
    margin-bottom: 15px;
}

/* Child selector */
nav > ul {
    list-style: none;
}

/* Multiple selectors */
h1, h2, h3 {
    font-family: Georgia, serif;
}

Specificity Rules


/* Specificity increases from top to bottom */

/* Universal selector */
* {
    margin: 0;
}

/* Element selector */
p {
    color: blue;
}

/* Class selector */
.text {
    color: red;
}

/* ID selector */
#unique {
    color: green;
}

/* Inline styles (highest specificity) */
<p style="color: purple;">Text</p>

/* !important (use sparingly) */
p {
    color: orange !important;
}

The Cascade

How Styles Cascade


/* Later styles override earlier ones */
p {
    color: blue;
}

p {
    color: red;  /* This wins */
}

/* More specific selectors override less specific ones */
p {
    color: blue;
}

.special p {
    color: red;  /* This wins due to higher specificity */
}

/* Inheritance */
body {
    font-family: Arial, sans-serif;  /* Inherited by all children */
    color: #333;                     /* Inherited by all children */
}

/* Some properties don't inherit by default */
button {
    border: none;  /* Not inherited */
}

Practice Exercise

Create a simple webpage with the following requirements:

  • External CSS file
  • Different types of selectors
  • Demonstrate specificity
  • Show inheritance

/* styles.css */
/* Base styles */
body {
    font-family: Arial, sans-serif;
    color: #333;
    line-height: 1.6;
}

/* Element selectors */
h1 {
    color: navy;
    text-align: center;
}

/* Class selectors */
.container {
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}

.highlight {
    background-color: yellow;
}

/* ID selector */
#main-header {
    background-color: #f4f4f4;
    padding: 1rem;
}

/* Combining selectors */
.container p {
    margin-bottom: 1.5rem;
}

/* Specificity example */
p {
    color: blue;
}

.container p {
    color: green;  /* This wins due to higher specificity */
}
