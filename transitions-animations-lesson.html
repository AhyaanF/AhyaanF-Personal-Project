CSS Transitions and Animations

Adding Motion and Interactivity

Learn how to create smooth transitions and complex animations:


/* Basic Transitions */
.button {
    background-color: blue;
    color: white;
    transition: all 0.3s ease;
}

.button:hover {
    background-color: darkblue;
    transform: scale(1.1);
}

/* Individual Property Transitions */
.card {
    transition-property: transform, opacity;
    transition-duration: 0.3s, 0.5s;
    transition-timing-function: ease-out, linear;
    transition-delay: 0s, 0.1s;
}

/* Shorthand Transition */
.element {
    transition: transform 0.3s ease-in-out,
                background-color 0.5s linear;
}

/* Basic Keyframe Animation */
@keyframes slide-in {
    0% {
        transform: translateX(-100%);
        opacity: 0;
    }
    100% {
        transform: translateX(0);
        opacity: 1;
    }
}

.sliding-element {
    animation: slide-in 1s ease-out forwards;
}

/* Multiple Keyframes */
@keyframes pulse {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); }
    100% { transform: scale(1); }
}

/* Animation Properties */
.animated-element {
    animation-name: pulse;
    animation-duration: 2s;
    animation-timing-function: ease-in-out;
    animation-delay: 0.5s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    animation-fill-mode: forwards;
    animation-play-state: running;
}

/* Animation Shorthand */
.element {
    animation: pulse 2s ease-in-out 0.5s infinite alternate;
}

/* Multiple Animations */
.complex-animation {
    animation: 
        slide-in 1s ease-out,
        fade-in 2s linear;
}

/* Transform Transitions */
.transform-element {
    transition: transform 0.3s;
}

.transform-element:hover {
    transform: rotate(45deg) scale(1.2) translateX(20px);
}

/* Timing Functions */
.timing-examples {
    transition-timing-function: linear;
    transition-timing-function: ease;
    transition-timing-function: ease-in;
    transition-timing-function: ease-out;
    transition-timing-function: ease-in-out;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
}

/* Animation Events */
.element {
    animation: bounce 1s;
}

.element.paused {
    animation-play-state: paused;
}

/* Performance Optimizations */
.optimized {
    transform: translateZ(0);
    will-change: transform;
}

Animation Concepts:

  • Transitions - Smooth property changes
  • Keyframes - Define animation sequences
  • Animation Properties - Control animation behavior
  • Transform Functions - Modify element appearance
  • Timing Functions - Control animation pacing
  • Performance - Optimize animations

Practice Exercise

Create animations that include:

  • Hover transitions with multiple properties
  • Keyframe animation with multiple steps
  • Transform combinations
  • Different timing functions
  • Animation sequence with delays
