HTML Attributes

Understanding HTML Attributes

HTML attributes provide additional information about elements and can modify their behavior. They are always specified in the start tag and usually come in name/value pairs.

Basic Attribute Syntax


<element attribute="value">content</element>

<!-- Examples -->
<img src="image.jpg" alt="A beautiful sunset" width="500">
<a href="https://example.com" target="_blank">Visit Example</a>
<p class="highlight" id="intro">This is a paragraph.</p>

Common HTML Attributes

Global Attributes


<!-- Class Attribute -->
<div class="container highlight">Multiple classes</div>

<!-- ID Attribute -->
<div id="header">Unique identifier</div>

<!-- Style Attribute -->
<p style="color: blue; font-size: 16px;">Inline styling</p>

<!-- Title Attribute -->
<span title="More info on hover">Hover me</span>

<!-- Data Attributes -->
<div data-user="john" data-role="admin">Custom data</div>

Element-Specific Attributes


<!-- Image Attributes -->
<img src="photo.jpg" alt="Description" width="300" height="200">

<!-- Link Attributes -->
<a href="page.html" target="_blank" rel="noopener">New Tab</a>

<!-- Form Attributes -->
<input type="text" name="username" required placeholder="Enter username">

<!-- Button Attributes -->
<button type="submit" disabled>Submit</button>

  • An image with proper src, alt, and size attributes
