Basic HTML Elements

Text Elements

HTML provides various elements for formatting text content. Let's explore the most common ones.

Text Formatting Elements


<!-- Headings -->
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>

<!-- Paragraphs -->
<p>This is a paragraph of text.</p>

<!-- Text Formatting -->
<strong>Bold text</strong>
<em>Italicized text</em>
<mark>Highlighted text</mark>
<u>Underlined text</u>

Links and Images

Learn how to add links and images to your web pages.

Links and Images Syntax


<!-- Basic Link -->
<a href="https://www.example.com">Visit Example.com</a>

<!-- Link to Another Page -->
<a href="about.html">About Us</a>

<!-- Basic Image -->
<img src="image.jpg" alt="Description of image">

<!-- Image with Size -->
<img src="logo.png" alt="Company Logo" width="200" height="100">

<!-- Linked Image -->
<a href="https://www.example.com">
    <img src="button.png" alt="Click here">
</a>

Practice Exercise

Create an HTML page that includes:

  • Different levels of headings
  • Paragraphs with formatted text
  • A link to your favorite website
  • An image with proper alt text

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Basic Elements Practice</title>
    </head>
    <body>
        <h1>My Favorite Things</h1>
        
        <h2>About Me</h2>
        <p>I am learning <strong>HTML</strong> and it's <em>very exciting</em>!</p>
        
        <h2>My Hobbies</h2>
        <p>I love <mark>web development</mark> and coding.</p>
        
        <h3>Favorite Website</h3>
        <a href="https://www.github.com">Visit GitHub</a>
        
        <h3>My Profile Picture</h3>
        <img src="profile.jpg" alt="My profile picture" width="200">
    </body>
</html>
