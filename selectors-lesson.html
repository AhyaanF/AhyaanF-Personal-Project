CSS Selectors and Specificity

Types of Selectors and Their Specificity

CSS selectors determine which elements to style and their specificity determines which styles take precedence:


/* Basic Selectors */
* { /* Universal selector */
    margin: 0;
    padding: 0;
}

div { /* Element selector */
    background: #f0f0f0;
}

.class-name { /* Class selector */
    color: blue;
}

#unique-id { /* ID selector */
    font-weight: bold;
}

/* Combinators */
div p { /* Descendant selector */
    margin: 10px;
}

div > p { /* Child selector */
    padding: 5px;
}

h2 + p { /* Adjacent sibling */
    font-size: 1.1em;
}

h2 ~ p { /* General sibling */
    line-height: 1.5;
}

/* Attribute Selectors */
[type="text"] { /* Exact match */
    border: 1px solid gray;
}

[class^="btn-"] { /* Starts with */
    padding: 10px;
}

[href*="example"] { /* Contains */
    color: purple;
}

[data-type$="post"] { /* Ends with */
    margin-bottom: 20px;
}

/* Pseudo-classes */
a:hover {
    text-decoration: underline;
}

li:first-child {
    font-weight: bold;
}

input:focus {
    outline: 2px solid blue;
}

/* Pseudo-elements */
p::first-line {
    font-size: 1.2em;
}

.quote::before {
    content: "❝";
}

/* Specificity Examples */
/* Specificity: 0-0-1 */
p { color: blue; }

/* Specificity: 0-1-0 */
.text { color: red; }

/* Specificity: 1-0-0 */
#header { color: green; }

/* Specificity: 0-2-1 */
.nav .list li { color: purple; }

/* Specificity: 1-1-1 */
#main .content p { color: orange; }

Selector Types:

  • Basic Selectors - Elements, classes, IDs
  • Combinators - Relationships between elements
  • Attribute Selectors - Element attributes
  • Pseudo-classes - Element states
  • Pseudo-elements - Parts of elements

Specificity Rules (from highest to lowest):

  1. Inline styles
  2. ID selectors
  3. Class, attribute, and pseudo-class selectors
  4. Element and pseudo-element selectors

Practice Exercise

Create CSS selectors that demonstrate:

  • Multiple types of basic selectors
  • At least two different combinators
  • Attribute selectors
  • Pseudo-classes and pseudo-elements
  • Different levels of specificity
