HTML Validation and Error Fixing

Common HTML Validation Issues

Learn to identify and fix common HTML validation errors:


<!-- ❌ Invalid HTML -->
<!DOCTYPE html>
<html>
<head>
    <title>Invalid Page</title>
    <meta charset="UTF-8">
<body>
    <h1>Missing Closing Tags
    <p>Unclosed paragraph
    <div>Nested incorrectly</p></div>
    <img src="photo.jpg">
    <a href=#>Invalid link</a>
</body>

<!-- ✅ Valid HTML -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Valid Page</title>
</head>
<body>
    <h1>Proper Structure</h1>
    <p>Properly closed paragraph</p>
    <div>
        <p>Correctly nested elements</p>
    </div>
    <img src="photo.jpg" alt="Description">
    <a href="#">Valid link</a>
</body>
</html>

<!-- Common Validation Issues -->
1. Missing required attributes:
<img src="photo.jpg"> ❌
<img src="photo.jpg" alt="Description"> ✅

2. Invalid attribute values:
<div class=myClass> ❌
<div class="myClass"> ✅

3. Improper nesting:
<b><i>Text</b></i> ❌
<b><i>Text</i></b> ✅

4. Missing closing tags:
<p>Text
<p>More text ❌

<p>Text</p>
<p>More text</p> ✅

Validation Checklist:

  • DOCTYPE Declaration - Must be first line
  • Required Attributes - Include all necessary attributes
  • Proper Nesting - Elements closed in correct order
  • Valid Values - Use correct attribute values
  • Complete Tags - All elements properly closed
  • Unique IDs - No duplicate ID attributes
  • Valid Characters - Use proper character encoding
