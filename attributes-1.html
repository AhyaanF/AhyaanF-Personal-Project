HTML Attributes

What are HTML Attributes?

Attributes provide additional information about HTML elements and can modify their behavior. They are always specified in the start tag.

Basic Attribute Syntax


<element attribute="value">content</element>

Common HTML Attributes

Link Attributes


<!-- href specifies the URL, target controls how the link opens -->
<a href="https://example.com" target="_blank">Open in new tab</a>

Image Attributes


<!-- src specifies image source, alt provides alternative text -->
<img src="cat.jpg" alt="Cute cat" width="300" height="200">

Form Input Attributes


<!-- type, name, placeholder, and required attributes -->
<input type="text" name="username" placeholder="Enter username" required>
<input type="password" name="password" minlength="8">

Global Attributes


<!-- class and id for styling and JavaScript -->
<div class="container" id="main">Content</div>

<!-- style for inline CSS -->
<p style="color: blue;">Blue text</p>

<!-- title for tooltips -->
<span title="More info">Hover me</span>

Practice Exercise

Create HTML elements using the following attributes:

  • A link that opens in a new tab
  • An image with width, height, and alt text
  • A text input with placeholder and required attributes
  • A div with both class and id attributes

<!-- Link in new tab -->
<a href="https://example.com" target="_blank">Visit Example</a>

<!-- Image with attributes -->
<img src="photo.jpg" alt="Beautiful landscape" width="800" height="600">

<!-- Input with attributes -->
<input type="text" placeholder="Enter your name" required>

<!-- Div with class and id -->
<div class="content-box" id="main-content">
    This is the main content
</div>
