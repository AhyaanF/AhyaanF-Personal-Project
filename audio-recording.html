Audio Recording

Audio Equipment

Learn about essential audio recording equipment and their proper usage.

Microphone Types


1. Dynamic Microphones
   - Durability: High
   - Sensitivity: Moderate
   - Best for: Loud sources
   - Examples: Shure SM58, SM7B

2. Condenser Microphones
   - Durability: Moderate
   - Sensitivity: High
   - Best for: Vocals, studio
   - Examples: Rode NT1, Blue Yeti

3. Lavalier Microphones
   - Durability: Varies
   - Sensitivity: High
   - Best for: Interview, speaking
   - Examples: Rode Wireless GO

4. Shotgun Microphones
   - Durability: High
   - Sensitivity: Directional
   - Best for: Field recording
   - Examples: Rode NTG3

Recording Techniques

Master the fundamentals of capturing clean, professional audio.

Recording Best Practices


1. Room Treatment
   - Minimize reflections
   - Control background noise
   - Use acoustic panels
   - Position away from walls

2. Microphone Placement
   - Distance: 6-12 inches
   - Angle: 45 degrees off-axis
   - Pop filter usage
   - Shock mount implementation

3. Gain Staging
   - Average level: -18dB
   - Peaks: -6dB
   - Headroom: -3dB
   - Avoid clipping

4. Monitoring
   - Use headphones
   - Check for interference
   - Monitor levels
   - Listen for artifacts

Audio Settings

Recording Parameters


1. Sample Rate
   - 44.1kHz: Standard
   - 48kHz: Video standard
   - 96kHz: High quality

2. Bit Depth
   - 16-bit: Standard
   - 24-bit: Professional
   - 32-bit float: Maximum quality

3. File Formats
   - WAV: Uncompressed
   - AIFF: Apple format
   - MP3: Compressed
   - AAC: Video audio

4. Buffer Size
   - 64 samples: Lowest latency
   - 128 samples: Balance
   - 256+ samples: High stability

Example Recording Setup:

1. Room Setup
   - Corner placement for minimal reflections
   - Acoustic panels on walls
   - Carpet or rug on floor
   - Door and window sealed

2. Equipment Setup
   - Main mic: Condenser with pop filter
   - Backup: Lavalier mic
   - Audio interface: 48kHz/24-bit
   - Headphones for monitoring

3. Recording Process
   - Set gain for -18dB average
   - Record 30 seconds of room tone
   - Test recording for each mic
   - Monitor levels continuously

4. Post-Check
   - Listen for background noise
   - Check for plosives
   - Verify stereo image
   - Confirm file format
