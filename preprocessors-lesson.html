CSS Preprocessors (SASS/SCSS)

Extending CSS with Preprocessors

Learn how to use SASS/SCSS to write more maintainable CSS:


// Variables
$primary-color: #3498db;
$secondary-color: #2ecc71;
$spacing-unit: 8px;
$font-stack: 'Arial', sans-serif;

// Nesting
.header {
    background: $primary-color;
    padding: $spacing-unit * 2;

    nav {
        display: flex;
        
        ul {
            list-style: none;
            
            li {
                margin-right: $spacing-unit;
                
                a {
                    color: white;
                    
                    &:hover {
                        text-decoration: underline;
                    }
                }
            }
        }
    }
}

// Mixins
@mixin flex-center {
    display: flex;
    justify-content: center;
    align-items: center;
}

@mixin media-query($width) {
    @media screen and (min-width: $width) {
        @content;
    }
}

// Using Mixins
.container {
    @include flex-center;
    
    @include media-query(768px) {
        flex-direction: row;
    }
}

// Functions
@function calculate-width($n) {
    @return $n * $spacing-unit;
}

// Extend/Inheritance
%button-base {
    padding: $spacing-unit $spacing-unit * 2;
    border-radius: 4px;
    border: none;
}

.button-primary {
    @extend %button-base;
    background: $primary-color;
}

.button-secondary {
    @extend %button-base;
    background: $secondary-color;
}

// Control Directives
@for $i from 1 through 3 {
    .col-#{$i} {
        width: $i * 25%;
    }
}

// Color Functions
.card {
    background: lighten($primary-color, 10%);
    border-color: darken($primary-color, 10%);
    
    &:hover {
        background: mix($primary-color, white, 80%);
    }
}

// Lists and Maps
$sizes: (
    small: $spacing-unit,
    medium: $spacing-unit * 2,
    large: $spacing-unit * 3
);

.element {
    padding: map-get($sizes, medium);
}

// Import Partials
@import 'variables';
@import 'mixins';
@import 'functions';

// Operators
.sidebar {
    width: (100% / 3);
    margin: $spacing-unit + 5px;
}

// Parent Selector
.link {
    color: blue;
    
    .sidebar & {
        color: red;
    }
}

Preprocessor Features:

  • Variables - Store reusable values
  • Nesting - Write nested selectors
  • Mixins - Reusable style blocks
  • Functions - Custom calculations
  • Inheritance - Share styles between selectors
  • Control Directives - Loops and conditions
