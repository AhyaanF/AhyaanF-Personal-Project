The CSS Box Model

Understanding the Box Model

The CSS box model is fundamental to layout and spacing in CSS. Every element is represented as a rectangular box with content, padding, border, and margin areas.

Box Model Components


/* Box Model Properties */
.box {
    /* Content */
    width: 200px;
    height: 100px;
    
    /* Padding */
    padding: 20px;
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;
    
    /* Border */
    border: 2px solid black;
    border-width: 2px;
    border-style: solid;
    border-color: black;
    
    /* Margin */
    margin: 20px;
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 10px;
    margin-left: 20px;
}

/* Shorthand Properties */
.box {
    /* All sides equal */
    padding: 20px;
    margin: 20px;
    
    /* Top/Bottom Left/Right */
    padding: 10px 20px;
    margin: 10px 20px;
    
    /* Top Right Bottom Left */
    padding: 10px 20px 15px 25px;
    margin: 10px 20px 15px 25px;
}

Box Sizing

Box Sizing Properties


/* Default Box Sizing */
.content-box {
    box-sizing: content-box;
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    /* Total width = 200px + 40px + 4px = 244px */
}

/* Border Box */
.border-box {
    box-sizing: border-box;
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    /* Total width = 200px (includes padding and border) */
}

/* Global Box Sizing Reset */
* {
    box-sizing: border-box;
}

Advanced Box Properties


/* Border Radius */
.rounded-box {
    border-radius: 10px;
    border-top-left-radius: 20px;
    border-top-right-radius: 10px;
    border-bottom-right-radius: 15px;
    border-bottom-left-radius: 25px;
}

/* Box Shadow */
.shadow-box {
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
    box-shadow: inset 0 0 10px rgba(0, 0, 0, 0.5);
    box-shadow: 
        2px 2px 5px rgba(0, 0, 0, 0.3),
        inset 0 0 10px rgba(0, 0, 0, 0.5);
}

/* Outline */
.outline-box {
    outline: 2px solid red;
    outline-offset: 5px;
}

    background-color: white;
