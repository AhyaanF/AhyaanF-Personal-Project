CSS Media Queries

Understanding Media Queries

Media queries allow you to apply CSS styles based on device characteristics like screen width, height, orientation, and more. They are fundamental to responsive web design.

Basic Media Query Syntax


/* Basic media query structure */
@media screen and (max-width: 768px) {
    /* Styles for screens up to 768px wide */
    .container {
        width: 100%;
        padding: 0 15px;
    }
}

/* Multiple conditions */
@media screen and (min-width: 768px) and (max-width: 1024px) {
    /* Styles for screens between 768px and 1024px */
    .container {
        width: 90%;
        margin: 0 auto;
    }
}

/* Print styles */
@media print {
    .no-print {
        display: none;
    }
}

Common Breakpoints

While breakpoints should be based on your content, here are common device-based breakpoints used in responsive design.

Standard Breakpoints


/* Mobile First Approach */
/* Base styles for mobile */
.element {
    width: 100%;
}

/* Tablet (Portrait) */
@media (min-width: 768px) {
    .element {
        width: 50%;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .element {
        width: 33.333%;
    }
}

/* Large Desktop */
@media (min-width: 1200px) {
    .element {
        width: 25%;
    }
}

/* High-DPI Screens */
@media (-webkit-min-device-pixel-ratio: 2),
       (min-resolution: 192dpi) {
    /* High-resolution styles */
    .logo {
        background-image: url('logo@2x.png');
    }
}

Media Features

Common Media Features


/* Width and Height */
@media (max-width: 600px) { }
@media (min-height: 800px) { }
@media (aspect-ratio: 16/9) { }

/* Orientation */
@media (orientation: landscape) {
    .sidebar {
        float: left;
        width: 30%;
    }
}

/* Display Quality */
@media (hover: hover) {
    .button:hover {
        background: #f0f0f0;
    }
}

/* Reduced Motion */
@media (prefers-reduced-motion: reduce) {
    .animation {
        animation: none;
        transition: none;
    }
}

/* Dark Mode */
@media (prefers-color-scheme: dark) {
    body {
        background: #333;
        color: #fff;
    }
}

Best Practices

Media Query Implementation


/* Mobile First Approach */
.card {
    /* Base styles for mobile */
    width: 100%;
    margin-bottom: 1rem;
}

/* Progressive Enhancement */
@media (min-width: 768px) {
    .card {
        width: calc(50% - 1rem);
        float: left;
        margin-right: 1rem;
    }
}

/* Avoid Device-Specific Names */
/* Bad */
@media (min-width: 768px) { /* tablet styles */ }

/* Good */
@media (min-width: 48em) { /* medium viewport styles */ }

/* Group Media Queries */
/* Option 1: At the end of your CSS */
/* styles.css */
/* Base styles */
/* ... */
/* All media queries */
@media (min-width: 768px) { /* ... */ }
@media (min-width: 1024px) { /* ... */ }

/* Option 2: Component-specific */
.component {
    /* Base styles */
}
@media (min-width: 768px) {
    .component {
        /* Responsive styles */
    }
}

Practice Exercise

Create a responsive navigation menu that:

  • Shows a hamburger menu on mobile
  • Expands to a horizontal menu on desktop
  • Has a dropdown submenu on larger screens
  • Adapts to different screen sizes smoothly

<!-- index.html -->
<nav class="main-nav">
    <button class="menu-toggle">☰</button>
    <ul class="nav-list">
        <li><a href="#">Home</a></li>
        <li class="has-dropdown">
            <a href="#">Products</a>
            <ul class="dropdown">
                <li><a href="#">Category 1</a></li>
                <li><a href="#">Category 2</a></li>
            </ul>
        </li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>

/* styles.css */
.main-nav {
    position: relative;
    background: #333;
    padding: 1rem;
}

.menu-toggle {
    display: block;
    background: none;
    border: none;
    color: white;
    font-size: 1.5rem;
    cursor: pointer;
}

.nav-list {
    display: none;
    list-style: none;
    margin: 0;
    padding: 0;
}

.nav-list.active {
    display: block;
}

.nav-list a {
    color: white;
    text-decoration: none;
    padding: 0.5rem 0;
    display: block;
}

.dropdown {
    display: none;
    list-style: none;
    padding-left: 1rem;
}

/* Tablet and up */
@media (min-width: 768px) {
    .menu-toggle {
        display: none;
    }
    
    .nav-list {
        display: flex;
        gap: 1rem;
    }
    
    .has-dropdown {
        position: relative;
    }
    
    .dropdown {
        position: absolute;
        top: 100%;
        left: 0;
        background: #333;
        min-width: 200px;
        padding: 0.5rem;
    }
    
    .has-dropdown:hover .dropdown {
        display: block;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .main-nav {
        padding: 1rem 2rem;
    }
    
    .nav-list {
        gap: 2rem;
    }
}
