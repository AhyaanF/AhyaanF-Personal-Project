CSS Transforms

2D Transforms

Learn how to manipulate elements in two-dimensional space.

Basic 2D Transforms


/* Translate */
.translate {
    transform: translate(50px, 100px);
    transform: translateX(50px);
    transform: translateY(100px);
}

/* Scale */
.scale {
    transform: scale(1.5);
    transform: scale(2, 1.5);
    transform: scaleX(2);
    transform: scaleY(1.5);
}

/* Rotate */
.rotate {
    transform: rotate(45deg);
    transform: rotate(-45deg);
}

/* Skew */
.skew {
    transform: skew(20deg);
    transform: skew(20deg, 10deg);
    transform: skewX(20deg);
    transform: skewY(10deg);
}

/* Multiple transforms */
.combined {
    transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}

/* Transform origin */
.origin {
    transform-origin: top left;
    transform-origin: 50% 50%;
    transform-origin: center center;
}

Transform Matrix


/* Matrix transformation */
.matrix {
    transform: matrix(1, 0, 0, 1, 0, 0); /* Identity matrix */
    transform: matrix(
        scaleX, skewY,
        skewX, scaleY,
        translateX, translateY
    );
}

/* Common matrix transformations */
.matrix-scale {
    transform: matrix(2, 0, 0, 2, 0, 0); /* Scale 2x */
}

.matrix-rotate {
    transform: matrix(0.7071, 0.7071, -0.7071, 0.7071, 0, 0); /* Rotate 45deg */
}

.matrix-translate {
    transform: matrix(1, 0, 0, 1, 100, 100); /* Translate 100px x,y */
}

3D Transforms

3D Transform Properties


/* 3D Transforms */
.transform-3d {
    /* Translate */
    transform: translate3d(x, y, z);
    transform: translateZ(50px);
    
    /* Scale */
    transform: scale3d(2, 2, 2);
    transform: scaleZ(2);
    
    /* Rotate */
    transform: rotate3d(1, 1, 1, 45deg);
    transform: rotateX(45deg);
    transform: rotateY(45deg);
    transform: rotateZ(45deg);
    
    /* Perspective */
    perspective: 1000px;
    perspective-origin: center center;
}

/* 3D Space */
.container-3d {
    transform-style: preserve-3d;
    perspective: 1000px;
}

.child-3d {
    backface-visibility: hidden;
    transform: rotateY(180deg);
}

/* Multiple 3D transforms */
.complex-3d {
    transform: 
        perspective(1000px)
        rotateX(45deg)
        rotateY(45deg)
        translateZ(100px);
}

Performance and Best Practices

Optimizing Transforms


/* Hardware acceleration */
.accelerated {
    transform: translateZ(0);
    will-change: transform;
}

/* Composite layers */
.composite {
    transform: translate3d(0, 0, 0);
    backface-visibility: hidden;
}

/* Transform vs position */
/* Better performance */
.better {
    transform: translate(100px, 100px);
}

/* Worse performance */
.worse {
    left: 100px;
    top: 100px;
}

/* Smooth animations */
.smooth {
    transition: transform 0.3s ease-out;
    will-change: transform;
}

/* Reduce repaints */
.efficient {
    transform: translate3d(0, 0, 0);
    backface-visibility: hidden;
    perspective: 1000px;
}

Practice Exercise

Create a 3D card flip effect that includes:

  • 3D perspective
  • Smooth rotation
  • Backface visibility handling
  • Performance optimizations

/* HTML Structure */
<div class="card-container">
    <div class="card">
        <div class="card-front">
            <h3>Front Side</h3>
            <p>Hover to flip</p>
        </div>
        <div class="card-back">
            <h3>Back Side</h3>
            <p>Hover to flip back</p>
        </div>
    </div>
</div>

/* CSS Styles */
.card-container {
    perspective: 1000px;
    width: 300px;
    height: 200px;
}

.card {
    position: relative;
    width: 100%;
    height: 100%;
    transform-style: preserve-3d;
    transition: transform 0.6s;
    cursor: pointer;
    will-change: transform;
}

.card:hover {
    transform: rotateY(180deg);
}

.card-front,
.card-back {
    position: absolute;
    width: 100%;
    height: 100%;
    backface-visibility: hidden;
    display: flex;
    flex-direction: column;
    align-items: center;
    justify-content: center;
    border-radius: 10px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

.card-front {
    background: linear-gradient(135deg, #6e8efb, #a777e3);
    color: white;
}

.card-back {
    background: linear-gradient(135deg, #a777e3, #6e8efb);
    color: white;
    transform: rotateY(180deg);
}

/* Performance Optimizations */
.card-container {
    transform: translateZ(0);
}

.card-front,
.card-back {
    transform: translateZ(1px);
    -webkit-font-smoothing: antialiased;
}

@media (prefers-reduced-motion: reduce) {
    .card {
        transition: none;
    }
    
    .card:hover {
        transform: none;
    }
}
